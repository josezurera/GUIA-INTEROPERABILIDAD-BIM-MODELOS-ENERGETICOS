# Glosario

| Término | Definición de trabajo |
|---|---|
| Modelo arquitectónico | Representación BIM orientada al diseño y documentación del edificio. |
| Modelo analítico | Abstracción geométrica y relacional utilizada para generar datos de cálculo. |
| Modelo energético | Conjunto de zonas, superficies, condiciones operacionales y sistemas utilizado por el motor de simulación. |
| Espacio térmico | Volumen sometido a condiciones térmicas y operacionales definidas. |
| Zona térmica | Agrupación de uno o varios espacios que comparten criterios de simulación. |
| Colindancia | Relación entre superficies o espacios que comparten un límite. |
| IFC | Esquema abierto de intercambio de información de construcción mantenido por buildingSMART. |
| MVD | Subconjunto o vista de implementación del esquema IFC para un uso concreto. |
