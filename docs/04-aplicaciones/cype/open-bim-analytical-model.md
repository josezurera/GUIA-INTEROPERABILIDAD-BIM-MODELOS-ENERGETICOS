# Open BIM Analytical Model

## Fuente inicial

Manual oficial `ob_analytical_01.pdf`, revisado el 11 de julio de 2026.

## Componentes del modelo

- Espacios.
- Superficies.
- Aristas.
- Sombras propias.
- Sombras remotas.
- Grupos de espacios.

El algoritmo puede utilizar recintos y elementos arquitectónicos del IFC para generar superficies, aristas, colindancias y uniones. El resultado es editable y requiere validación.
