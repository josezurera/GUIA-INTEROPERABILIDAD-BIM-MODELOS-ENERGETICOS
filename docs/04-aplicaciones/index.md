# Aplicaciones de análisis

Cada módulo documentará formatos admitidos, proceso de importación, información reconocida, simplificaciones, actualización, limitaciones y checklist de validación.

OpenStudio y EnergyPlus constituyen el flujo abierto de simulación energética de la guía. Se documentan como componentes relacionados pero distintos: OpenStudio prepara y automatiza el modelo y EnergyPlus ejecuta el cálculo.
