# Registro de incidencias

| Campo | Descripción |
|---|---|
| Identificador | Código único de la incidencia. |
| Versión Revit | Versión exacta del modelo fuente. |
| Exportador | Versión y configuración IFC. |
| Receptor | Aplicación y versión. |
| Síntoma | Comportamiento observado. |
| Origen | Revit, IFC, modelo analítico o motor. |
| Acción | Corrección aplicada. |
| Evidencia | Capturas, archivos o resultados. |
| Estado | Abierta, validada o descartada. |
