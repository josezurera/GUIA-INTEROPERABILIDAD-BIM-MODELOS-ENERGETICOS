# Del modelo arquitectónico al modelo energético

La geometría arquitectónica es la fuente, pero no constituye por sí sola un modelo energético. El proceso requiere interpretar recintos, superficies de intercambio, huecos, contactos, sombras y agrupaciones térmicas.

## Capas del proceso

1. Modelo nativo de Revit.
2. Modelo de intercambio IFC.
3. Modelo analítico.
4. Modelo energético del motor de cálculo.
5. Resultados y documentación reglamentaria.
