# Checklist preliminar de Revit

- [ ] Versión de Revit y exportador registradas.
- [ ] Vista específica de exportación creada.
- [ ] Niveles analíticamente relevantes identificados.
- [ ] Coordenadas, ubicación y norte verificados.
- [ ] Habitaciones colocadas y cerradas.
- [ ] Cálculo de áreas y volúmenes activado.
- [ ] Elementos `Room Bounding` revisados.
- [ ] Envolvente exterior/interior clasificada.
- [ ] Huecos correctamente hospedados.
- [ ] Fases y vinculados revisados.
- [ ] Elementos decorativos o innecesarios excluidos.
- [ ] Exportación piloto validada.
