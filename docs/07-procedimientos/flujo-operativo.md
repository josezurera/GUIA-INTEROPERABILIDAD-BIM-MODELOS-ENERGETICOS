# Flujo operativo

1. Definir el uso energético del modelo.
2. Preparar espacios y envolvente en Revit.
3. Ejecutar QA/QC del modelo fuente.
4. Exportar un IFC piloto.
5. Validar geometría y estructura del IFC.
6. Generar o reconstruir el modelo analítico.
7. Validar espacios, superficies, adyacencias y sombras.
8. Completar el modelo energético.
9. Registrar incidencias y retroalimentar el modelo fuente.
10. Cerrar una versión reproducible.
