# IFC de ensayo

Coloque aquí archivos `.ifc` no confidenciales destinados a regresión. GitHub los validará contra las especificaciones de `config/ids/` y publicará los informes como artefactos de la acción **Validar IFC con IDS**.

No deben incorporarse modelos de proyecto sin autorización. Para archivos privados, ejecute el validador localmente y conserve los informes fuera del repositorio.

`eem-minimo-conforme.ifc` es un archivo sintético sin geometría utilizado únicamente para comprobar el funcionamiento del validador y del IDS mínimo.
